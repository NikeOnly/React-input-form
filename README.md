# React-input-form

To build and start the project you need:
 1) Execute in terminal (or command prompt)
```
    npm install
    npm run build
```
2) Go to `target` folder and open `index.html` with browser


To launch the watcher you need:
 1) Execute in terminal (or command prompt)
```
    npm install
    npm run watch
```
2) Go to `target` folder and open `index.html` with browser
